## 📖 About The Project

This repository contains the source code for the official website of the **Arab Innovation Network (AIN) - An-Najah National University (NNU) Chapter**.

Established at An-Najah in 2011, AIN is a registered student society designed to foster innovation, healthy competition, and technical growth among Arab students. Our branch works closely with the An-Najah Innovation Park and the Deanship of Student Affairs to organize annual conferences, hackathons, and exhibitions that bridge the gap between academic learning and real-world entrepreneurship.

**🌍 Live Website:** [ainnnu.vercel.app](https://ainnnu.vercel.app/index.html)

---

## 🎯 Our Mission & Vision

* **Vision:** To be the leading network in fostering innovation in science and technology in the Arab world.
* **Mission:** Attracting, motivating, and developing young innovators' skills, while improving the level of education by engaging with youth to generate enthusiasm around innovation and entrepreneurship.

## ✨ Key Features & Activities

The website showcases the core activities and initiatives of our student branch, including:

* 🏆 **AIN Innovators Hackathon:** Highlighting our flagship 24-hour rapid problem-solving events for engineering and IT students.
* 🎓 **SEED Scholarship:** Information regarding the Scholarship for Excellence in Education and Development, supporting outstanding students.
* 🔭 **Tech Exhibitions:** Showcasing massive student-led events like the *"An Eye on the Future"* exhibition.
* 👥 **Team Roster:** Introducing our dedicated administrative board, volunteers, and university supervisors.

## 🚀 Local Setup & Installation

To run this website locally on your machine, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AIN-Team/AIN-Website.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd AIN-Website
   ```

3. **Open in your browser:**

   You can simply open `index.html` directly in any browser, or use a local development server for a better experience:

   ```bash
   # Option A: Using VS Code Live Server extension
   # Right-click index.html → "Open with Live Server"

   # Option B: Using Python
   python -m http.server 8000
   # Then visit http://localhost:8000

   # Option C: Using Node.js (npx)
   npx serve .
   # Then visit http://localhost:3000
   ```

> ✅ No build step, no dependencies to install — it just works!

## 🌍 Deployment

The website is automatically deployed via **Vercel** on every push to the `main` branch.

| Environment | URL |
|-------------|-----|
| Production | [https://ainnnu.vercel.app](https://ainnnu.vercel.app/index.html) |

To deploy your own fork:
1. Fork this repository
2. Go to [vercel.com](https://vercel.com) and import your fork
3. Vercel auto-detects it as a static site — no configuration needed
4. Every push to `main` triggers an automatic redeployment
